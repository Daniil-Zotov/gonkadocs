gonkadocs.ai
Единый портал с документацией gonka.ai

gonka-ai - официальная документация протокола, документация от крманды разработчиков, дополнительные материалы от сообщества 
community - вся документация сообщества, дорожная карта, документы комитетов, документы foundation 
proposals - дашборд proposals, связанные документы, отчеты, аудиты от сообщества, мониторинг транзакций, ссылки на AMA
